葡萄飞盘

Grape Frisbee

葡萄飞盘（Grape Frisbee）是国内首个专注于飞盘活动的发布和参与平台。我们致力于为飞盘爱好者提供一站式的活动组织、参与和社交体验，帮助他们更方便地找到附近的活动，提升运动技能，并与志同道合的朋友分享运动乐趣。

核心功能

🥏 活动发布与报名

支持俱乐部发布定制活动，开通保险，用户数据分析和管理账号，降低活动发布门槛，让每个人都可以轻松地发布和定制优质活动。

🥏 赛事规则与技术辅助

提供飞盘领域的赛事规则、技术指导咨询、飞盘战术板工具等，帮助主办方更好地组织活动及提升运动水平。用户可以科学准确地了解这项运动，从而提升自己的竞技水平。

🥏 活动筛选与报名

用户可以根据地理位置和时间筛选活动，实现快速报名，轻松参与感兴趣的飞盘活动。

🥏 运动社交

用户可根据关联活动发布相关照片，结合飞盘运动进行兴趣社交，增进彼此间的友谊与交流，共同提高飞盘技术。
